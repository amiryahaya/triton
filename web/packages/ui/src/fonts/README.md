# Fonts

Self-hosted `.woff2` files. Licences retained in each font's upstream
repository — do not strip licence headers or repackage.

- **Inter** — [rsms.me/inter](https://rsms.me/inter) · SIL Open Font Licence 1.1
- **Geist** — [vercel.com/font](https://vercel.com/font) · SIL Open Font Licence 1.1
- **JetBrains Mono** — [jetbrains.com/mono](https://www.jetbrains.com/lp/mono/) · Apache 2.0

Do NOT convert, subset, or re-optimise these files unless every weight is
re-verified against the source distribution and any new files are re-licenced
per upstream.

Weights present:

| Family           | Weights                           |
|------------------|-----------------------------------|
| Inter            | 400 · 500 · 600 · 700             |
| Geist            | 500 · 600 · 700                   |
| JetBrains Mono   | 400 · 500                         |

If you need additional weights, download from upstream and add matching
`@font-face` blocks in `fonts.css`.
