Default window — which feels right?

All options remain switchable via the filter chips above; this is only the first view the operator sees.

A

30 days

Strict "act this sprint" view. Usually a very short list. Best if operators only want immediate action items on this page, and are happy clicking "90 days" to widen.

B

90 days ⭐

Matches browser and OS cert warning windows (e.g. macOS Keychain, Chrome). Gives operators a quarter's worth of runway to plan renewals. My recommendation.

C

180 days

Long-range planning view. Useful for government ops with long procurement cycles, but most of the list at any given moment is "not urgent."

D

All, sorted by soonest

Show every cert Triton knows about, sorted ascending by expiry. Lets operators see the full picture at once. Can get long for big orgs with many certs, but natural grouping via the status badges means the important ones are at the top.
