# Manage Server B2.2 — Scanner Orchestrator + mTLS Push Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Manage Server actually scan on-prem: own zones/hosts/scan-jobs/credentials, run `pkg/scanner` in-process via a worker pool, push results to Report over mTLS, mint agent certs from its own CA, and replace Report's 501 stub with real bundle issuance.

**Architecture:** Two listeners on the single `triton-manage` binary. `:8082` keeps the B1 admin listener (JWT) and gains Manage-native CRUD for zones + hosts + orchestrator + agent enrolment. `:8443` is a new TLS listener using Manage's own CA as the trust anchor — agents phone home here using certs Manage mints at enrol-time. A worker-pool goroutine group claims `queued` scan jobs from Postgres via `FOR UPDATE SKIP LOCKED`, drives `pkg/scanner` in-process, writes completed results into `scan_results_queue`, and a separate drain goroutine POSTs those rows to Report over mTLS with exponential backoff. Licence enforcement sits at three request-entry call-sites (seats, hosts, scans).

**Tech Stack:** Go 1.25, chi/v5 router, pgx/v5 pool (already injected in B2.1), `pkg/scanner.Engine`, `crypto/x509` for CA + agent cert minting, `crypto/tls` `RequireAndVerifyClientCert` for the :8443 listener, existing `internal/license.Guard` + `UsagePusher` for licence state, `httptest.Server` for integration tests.

**Assumptions / spec reconciliations (read before starting):**

1. **Two table namespaces co-exist in `triton_manage`.** Batch A creates Manage-native tables with a `manage_` prefix (`manage_scan_jobs`, `manage_zones`, `manage_hosts`, `manage_zone_memberships`, `manage_ca`, `manage_agent_cert_revocations`, `manage_push_creds`, `manage_license_state`, `manage_scan_results_queue`, `manage_scan_results_dead_letter`). Report-side tables (created by `store.Migrate(pool)`) sit unused. This avoids collisions with the Report monolithic migration set (`scan_jobs`, `scans`, `credentials_profiles`, …). Schema-version tracking uses a new `manage_schema_version` table so the two migration streams don't collide on the existing `schema_version` row.
2. **The 4 `pkg/server/*` packages (`scanjobs`, `credentials`, `discovery`, `agentpush`) are NOT mounted on Manage.** Spec §4 lists them for re-mount but their domain models (engine-claimed jobs, per-org credentials scoped by JWT org claim, Report's `agent_push_jobs`) do not match Manage's in-process orchestrator + Manage-CA-minted-agent model. Manage gets NATIVE packages that implement §5/§6/§7 semantics over `manage_*` tables. If a future change wants to reconcile and literally reuse the Report packages, that's a separate refactor. This plan delivers working B2.2 with native packages.
3. **Licence v2 fields (`features.manage`, `limits.seats/hosts/agents/scans`).** Use `guard.HasFeature("manage")`, `guard.LimitCap("<metric>","<window>")`, `guard.CurrentUsage(…)`, `guard.SoftBufferCeiling(…)` (all exist today per `internal/license/features*.go`). The licence validated by `/setup/license` must carry `features.manage=true`, else setup rejects. B1 already enforces this.
4. **Report's engine CA is reused for Manage enrolment.** Batch G's Report-side `pkg/server/manage_enrol/` package forks minimal CA primitives from `pkg/server/engine` (it signs a new leaf cert with the existing engine CA). No new CA on Report.
5. **TOFU for Report endpoint on Manage.** The Manage → Report `/scans` push trusts Report's CA via the `ca.crt` embedded in the push bundle Report returns from `/admin/enrol/manage`. The mTLS handshake uses that `RootCAs` pool; no TOFU-on-first-push.

## File Structure

**New packages (Manage side, under `pkg/manageserver/`):**
- `zones/` — CRUD for zones (admin handlers, postgres store, types, routes).
- `hosts/` — CRUD for hosts (admin handlers, postgres store, types, routes).
- `scanjobs/` — Manage-native scan-job enqueue + orchestrator worker pool.
- `scanresults/` — `scan_results_queue` writer + drain goroutine + HTTP client to Report.
- `ca/` — Manage CA key lifecycle + agent cert signing + bundle tarball.
- `agents/` — Admin enrol + revoke + gateway phone-home handlers, mtlsCNAuth middleware.
- `licenseenf/` — Licence guard call-sites used by zones/hosts/scanjobs handlers.

**Extended packages (Manage side, modifying existing B1 files):**
- `pkg/manageserver/server.go` — Wires the new packages, starts `:8443` listener, spawns orchestrator + drain goroutines.
- `pkg/manageserver/handlers_setup.go` — After `/setup/license` succeeds, calls Report's `/admin/enrol/manage` and persists the push bundle.
- `pkg/manageserver/config.go` — New fields: `GatewayListen`, `GatewayTLSCert`/`Key` (optional — Manage CA is the trust root), `ReportServer`, `Parallelism`.
- `pkg/managestore/migrations.go` — Appends all new `manage_*` tables.

**New packages (Report side):**
- `pkg/server/manage_enrol/` — `/api/v1/admin/enrol/manage` replaces the 501 stub (§7.3). Fork CA primitives from `pkg/server/engine`.

**Modified packages (Report side):**
- `pkg/server/engine/mtls_middleware.go` — Extend CN prefix matching: accept `manage:<…>` in addition to `engine:<…>`.
- `pkg/server/handlers_admin.go` (or the current location of the 501 stub) — Remove the 501, route to `manage_enrol` package.

**Infrastructure:**
- `Containerfile.manageserver` — Expose `:8443`.
- `compose.yaml` — Add `"8443:8443"` to manageserver service, env var `TRITON_MANAGE_GATEWAY_LISTEN=:8443`.
- `.env.example` — Document `TRITON_MANAGE_GATEWAY_LISTEN`, `TRITON_MANAGE_PARALLELISM`, `TRITON_MANAGE_REPORT_URL`.

**Tests:**
- Unit tests sit next to each source file (`ca_test.go`, `drain_test.go`, …) with `//go:build integration` for DB-touching tests to match the project's existing split.
- `test/integration/manage_e2e_scan_flow_test.go` — the full end-to-end happy path.

```go
// pkg/manageserver/zones/store.go
package zones

import (
    "context"
    "errors"
    "github.com/google/uuid"
)

var ErrNotFound = errors.New("zones: not found")

type Store interface {
    Create(ctx context.Context, z Zone) (Zone, error)
    Get(ctx context.Context, id uuid.UUID) (Zone, error)
    List(ctx context.Context) ([]Zone, error)
    Update(ctx context.Context, z Zone) (Zone, error)
    Delete(ctx context.Context, id uuid.UUID) error
    Count(ctx context.Context) (int64, error)
}
```

```go
// pkg/manageserver/zones/postgres.go
package zones

import (
    "context"
    "errors"

    "github.com/google/uuid"
    "github.com/jackc/pgx/v5"
    "github.com/jackc/pgx/v5/pgxpool"
)

type PostgresStore struct{ pool *pgxpool.Pool }

func NewPostgresStore(pool *pgxpool.Pool) *PostgresStore { return &PostgresStore{pool: pool} }

var _ Store = (*PostgresStore)(nil)

func (s *PostgresStore) Create(ctx context.Context, z Zone) (Zone, error) {
    err := s.pool.QueryRow(ctx,
        `INSERT INTO manage_zones (name, description) VALUES ($1,$2)
         RETURNING id, created_at, updated_at`,
        z.Name, z.Description,
    ).Scan(&z.ID, &z.CreatedAt, &z.UpdatedAt)
    return z, err
}

func (s *PostgresStore) Get(ctx context.Context, id uuid.UUID) (Zone, error) {
    var z Zone
    err := s.pool.QueryRow(ctx,
        `SELECT id, name, description, created_at, updated_at FROM manage_zones WHERE id=$1`, id,
    ).Scan(&z.ID, &z.Name, &z.Description, &z.CreatedAt, &z.UpdatedAt)
    if errors.Is(err, pgx.ErrNoRows) {
        return Zone{}, ErrNotFound
    }
    return z, err
}

func (s *PostgresStore) List(ctx context.Context) ([]Zone, error) {
    rows, err := s.pool.Query(ctx, `SELECT id, name, description, created_at, updated_at FROM manage_zones ORDER BY name`)
    if err != nil { return nil, err }
    defer rows.Close()
    out := []Zone{}
    for rows.Next() {
        var z Zone
        if err := rows.Scan(&z.ID, &z.Name, &z.Description, &z.CreatedAt, &z.UpdatedAt); err != nil { return nil, err }
        out = append(out, z)
    }
    return out, rows.Err()
}

func (s *PostgresStore) Update(ctx context.Context, z Zone) (Zone, error) {
    err := s.pool.QueryRow(ctx,
        `UPDATE manage_zones SET name=$1, description=$2, updated_at=NOW() WHERE id=$3
         RETURNING id, name, description, created_at, updated_at`,
        z.Name, z.Description, z.ID,
    ).Scan(&z.ID, &z.Name, &z.Description, &z.CreatedAt, &z.UpdatedAt)
    if errors.Is(err, pgx.ErrNoRows) { return Zone{}, ErrNotFound }
    return z, err
}

func (s *PostgresStore) Delete(ctx context.Context, id uuid.UUID) error {
    tag, err := s.pool.Exec(ctx, `DELETE FROM manage_zones WHERE id=$1`, id)
    if err != nil { return err }
    if tag.RowsAffected() == 0 { return ErrNotFound }
    return nil
```

```go
// ClaimNext uses FOR UPDATE SKIP LOCKED to let N workers pull concurrently.
func (s *PostgresStore) ClaimNext(ctx context.Context, workerID string) (Job, bool, error) {
    row := s.pool.QueryRow(ctx, `
        UPDATE manage_scan_jobs
           SET status = 'running',
               started_at = NOW(),
               running_heartbeat_at = NOW(),
               worker_id = $1
         WHERE id = (
            SELECT id FROM manage_scan_jobs
             WHERE status = 'queued'
             ORDER BY enqueued_at
             LIMIT 1
             FOR UPDATE SKIP LOCKED
         )
         RETURNING `+jobSelectCols,
        workerID)
    j, err := scanJob(row)
    if errors.Is(err, pgx.ErrNoRows) { return Job{}, false, nil }
    if err != nil { return Job{}, false, err }
    return j, true, nil
}

// Enqueue: expand zones → hosts; insert N rows in one tx.
func (s *PostgresStore) Enqueue(ctx context.Context, req EnqueueReq) ([]Job, error) {
    if err := s.checkSaturation(ctx); err != nil { return nil, err }

    tx, err := s.pool.BeginTx(ctx, pgx.TxOptions{})
    if err != nil { return nil, err }
    defer tx.Rollback(ctx)

    // expand zones to hosts
    rows, err := tx.Query(ctx,
        `SELECT id FROM manage_hosts WHERE zone_id = ANY($1) AND hostname LIKE $2`,
        req.ZoneIDs, sqlGlob(req.HostFilter))
    if err != nil { return nil, err }
    var hostIDs []uuid.UUID
    for rows.Next() {
        var id uuid.UUID
        if err := rows.Scan(&id); err != nil { return nil, err }
        hostIDs = append(hostIDs, id)
    }
    rows.Close()
    if err := rows.Err(); err != nil { return nil, err }

    out := make([]Job, 0, len(hostIDs))
    for _, hid := range hostIDs {
        var j Job
        j.TenantID = req.TenantID; j.Profile = req.Profile; j.CredentialsRef = req.CredentialsRef
        err := tx.QueryRow(ctx, `
            INSERT INTO manage_scan_jobs (tenant_id, zone_id, host_id, profile, credentials_ref)
            SELECT $1, h.zone_id, h.id, $2, $3 FROM manage_hosts h WHERE h.id=$4
            RETURNING `+jobSelectCols, req.TenantID, string(req.Profile), req.CredentialsRef, hid,
        ).Scan(&j.ID, &j.TenantID, &j.ZoneID, &j.HostID, &j.Profile, &j.CredentialsRef,
               &j.Status, &j.CancelRequested, &j.WorkerID, &j.EnqueuedAt, &j.StartedAt,
               &j.FinishedAt, &j.RunningHeartbeatAt, &j.ProgressText, &j.ErrorMessage)
        if err != nil { return nil, err }
        out = append(out, j)
    }
    return out, tx.Commit(ctx)
}

func (s *PostgresStore) checkSaturation(ctx context.Context) error {
    d, err := s.QueueDepth(ctx)
    if err != nil { return err }
    if d >= 10_000 { return ErrQueueSaturated }
    return nil
}

func sqlGlob(f string) string {
    if f == "" { return "%" }
    // simplistic glob: * → %
    return strings.ReplaceAll(f, "*", "%")
}
```

4. On success, write the result into `scan_results_queue` + call `Complete(job)`.
5. On failure, `Fail(job, err.Error())`.

```go
type ResultEnqueuer interface {

type OrchestratorConfig struct {

    Parallelism int

type Orchestrator struct{ cfg OrchestratorConfig }

func NewOrchestrator(cfg OrchestratorConfig) *Orchestrator {
    if cfg.Parallelism <= 0 { cfg.Parallelism = 10 }
    if cfg.Parallelism > 50 { cfg.Parallelism = 50 }
    if cfg.HeartbeatInterval == 0 { cfg.HeartbeatInterval = 60 * time.Second }
    if cfg.PollInterval == 0 { cfg.PollInterval = 2 * time.Second }
    if cfg.ReapAfter == 0 { cfg.ReapAfter = 5 * time.Minute }
    if cfg.ScanFunc == nil { cfg.ScanFunc = defaultScanFunc }
    return &Orchestrator{cfg: cfg}
}

// Run starts N workers + a reaper goroutine. Blocks until ctx.Done.
func (o *Orchestrator) Run(ctx context.Context) {
    var wg sync.WaitGroup
    for i := 0; i < o.cfg.Parallelism; i++ {
        wg.Add(1)
        go func(i int) {
            defer wg.Done()
            o.workerLoop(ctx, workerID(i))
        }(i)
    }
    wg.Add(1)
    go func() { defer wg.Done(); o.reaperLoop(ctx) }()
    wg.Wait()
}

func workerID(i int) string {
    hn, _ := os.Hostname()
    return fmt.Sprintf("%s:%d:%d", hn, os.Getpid(), i)
}

func (o *Orchestrator) workerLoop(ctx context.Context, wid string) {
    for {
        select { case <-ctx.Done(): return; default: }

        j, ok, err := o.cfg.Store.ClaimNext(ctx, wid)
        if err != nil {
            log.Printf("orchestrator: claim error: %v", err)
            time.Sleep(o.cfg.PollInterval); continue
        }
        if !ok {
            select {
            case <-time.After(o.cfg.PollInterval):
            case <-ctx.Done(): return
            }
            continue
        }

        o.runOneJob(ctx, j)
    }
}

func (o *Orchestrator) runOneJob(parent context.Context, j Job) {
    jobCtx, cancelJob := context.WithCancel(parent)
    defer cancelJob()

    // heartbeat + cancel watcher in a helper goroutine
    hbDone := make(chan struct{})
    go func() {
        t := time.NewTicker(o.cfg.HeartbeatInterval); defer t.Stop()
        for {
            select {
            case <-hbDone: return
            case <-t.C:
                _ = o.cfg.Store.Heartbeat(jobCtx, j.ID, j.ProgressText)
                if req, _ := o.cfg.Store.IsCancelRequested(jobCtx, j.ID); req {
                    cancelJob()
                }
            }
        }
    }()

    scan, err := o.cfg.ScanFunc(jobCtx, j)
    close(hbDone)

    if ctxErr := jobCtx.Err(); ctxErr != nil {
        // either parent shut down, or cancelJob was fired from the watcher
        if parent.Err() == nil {
            _ = o.cfg.Store.Cancel(jobCtx, j.ID)
        }
        return
    }
    if err != nil {
        _ = o.cfg.Store.Fail(parent, j.ID, err.Error())
        return
    }
    if err := o.cfg.ResultStore.Enqueue(parent, j.ID, "manage", o.cfg.SourceID, scan); err != nil {
        _ = o.cfg.Store.Fail(parent, j.ID, "enqueue result: "+err.Error())
        return
    }
    _ = o.cfg.Store.Complete(parent, j.ID)
}

func (o *Orchestrator) reaperLoop(ctx context.Context) {
    t := time.NewTicker(60 * time.Second); defer t.Stop()
    for {
        select {
        case <-ctx.Done(): return
        case <-t.C:
            if n, err := o.cfg.Store.ReapStale(ctx, o.cfg.ReapAfter); err == nil && n > 0 {
                log.Printf("orchestrator: reaped %d stale jobs", n)
            }
        }
    }
}

    return nil, fmt.Errorf("no scan func configured")
```

```go
func (s *PostgresStore) Enqueue(ctx context.Context, scanJobID uuid.UUID, sourceType string, sourceID uuid.UUID, scan *model.ScanResult) error {
    // Attach provenance inline:
    envelope := map[string]any{
        "scan":         scan,
        "submitted_by": map[string]any{"type": sourceType, "id": sourceID.String()},
    }
    payload, err := json.Marshal(envelope)
    if err != nil { return err }

    _, err = s.pool.Exec(ctx, `
        INSERT INTO manage_scan_results_queue (scan_job_id, source_type, source_id, payload_json)
        VALUES ($1, $2, $3, $4)`, scanJobID, sourceType, sourceID, payload)
    return err
}

func (s *PostgresStore) ClaimDue(ctx context.Context, limit int) ([]QueueRow, error) {
    rows, err := s.pool.Query(ctx, `
        SELECT id, scan_job_id, source_type, source_id, payload_json,
               enqueued_at, next_attempt_at, attempt_count, COALESCE(last_error,'')
          FROM manage_scan_results_queue
         WHERE next_attempt_at <= NOW() AND attempt_count < 10
         ORDER BY enqueued_at
         LIMIT $1`, limit)
    if err != nil { return nil, err }
    defer rows.Close()
    var out []QueueRow
    for rows.Next() {
        var q QueueRow
        if err := rows.Scan(&q.ID, &q.ScanJobID, &q.SourceType, &q.SourceID, &q.PayloadJSON,
            &q.EnqueuedAt, &q.NextAttemptAt, &q.AttemptCount, &q.LastError); err != nil { return nil, err }
        out = append(out, q)
    }
    return out, rows.Err()
}

// Defer: bump attempt_count, shift next_attempt_at.
func (s *PostgresStore) Defer(ctx context.Context, id uuid.UUID, nextAttempt time.Time, errMsg string) error {
    _, err := s.pool.Exec(ctx, `
        UPDATE manage_scan_results_queue
           SET attempt_count = attempt_count+1, next_attempt_at=$2, last_error=$3
         WHERE id=$1`, id, nextAttempt, errMsg)
    return err
}

// DeadLetter: copy to dead-letter + delete.
func (s *PostgresStore) DeadLetter(ctx context.Context, id uuid.UUID, reason string) error {
    tx, err := s.pool.BeginTx(ctx, pgx.TxOptions{}); if err != nil { return err }
    defer tx.Rollback(ctx)
    _, err = tx.Exec(ctx, `
        INSERT INTO manage_scan_results_dead_letter
            (id, scan_job_id, source_type, source_id, payload_json, enqueued_at, attempt_count, last_error, dead_letter_reason)
        SELECT id, scan_job_id, source_type, source_id, payload_json, enqueued_at, attempt_count, COALESCE(last_error,''), $2
        FROM manage_scan_results_queue WHERE id=$1`, id, reason)
    if err != nil { return err }
    if _, err := tx.Exec(ctx, `DELETE FROM manage_scan_results_queue WHERE id=$1`, id); err != nil { return err }
    return tx.Commit(ctx)
}
```

```go
package scanresults

import (
    "bytes"
    "context"
    "crypto/tls"
    "crypto/x509"
    "fmt"
    "log"
    "net/http"
    "time"

    "github.com/jackc/pgx/v5/pgxpool"
)

type DrainConfig struct {
    Store     Store
    Pool      *pgxpool.Pool // used to load manage_push_creds on startup + refresh
    ReportURL string
    Client    *http.Client
    Batch     int
    Interval  time.Duration
}

type Drain struct{ cfg DrainConfig }

func NewDrain(cfg DrainConfig) *Drain {
    if cfg.Batch == 0 { cfg.Batch = 100 }
    if cfg.Interval == 0 { cfg.Interval = 5 * time.Second }
    return &Drain{cfg: cfg}
}

func (d *Drain) Run(ctx context.Context) {
    t := time.NewTicker(d.cfg.Interval); defer t.Stop()
    for {
        select {
        case <-ctx.Done(): return
        case <-t.C:
            d.drainOnce(ctx)
        }
    }
}

func (d *Drain) drainOnce(ctx context.Context) {
    rows, err := d.cfg.Store.ClaimDue(ctx, d.cfg.Batch)
    if err != nil {
        log.Printf("drain: claim due: %v", err); return
    }
    for _, row := range rows {
        if err := d.push(ctx, row); err != nil {
            next := nextAttemptAt(row.AttemptCount + 1)
            _ = d.cfg.Store.Defer(ctx, row.ID, next, err.Error())
            continue
        }
        _ = d.cfg.Store.Delete(ctx, row.ID)
    }
}

func (d *Drain) push(ctx context.Context, r QueueRow) error {
    req, err := http.NewRequestWithContext(ctx, "POST", d.cfg.ReportURL+"/api/v1/scans", bytes.NewReader(r.PayloadJSON))
    if err != nil { return err }
    req.Header.Set("Content-Type", "application/json")
    resp, err := d.cfg.Client.Do(req)
    if err != nil { return err }
    defer resp.Body.Close()
    switch {
    case resp.StatusCode/100 == 2:
        return nil
    case resp.StatusCode/100 == 4:
        // non-retryable; dead-letter
        _ = d.cfg.Store.DeadLetter(ctx, r.ID, fmt.Sprintf("HTTP %d (non-retryable)", resp.StatusCode))
        return nil // swallowed — already handled
    default:
        return fmt.Errorf("HTTP %d", resp.StatusCode)
    }
}

func nextAttemptAt(attempt int) time.Time {
    // 10s, 20s, 40s, 80s, 160s, 300s (cap)
    secs := 10 << (attempt - 1)
    if secs > 300 { secs = 300 }
    return time.Now().Add(time.Duration(secs) * time.Second)
}

// BuildHTTPClient: mTLS client trusting Report's CA, using client cert+key from manage_push_creds.
func BuildHTTPClient(creds PushCreds) (*http.Client, error) {
    cert, err := tls.X509KeyPair([]byte(creds.ClientCertPEM), []byte(creds.ClientKeyPEM))
    if err != nil { return nil, fmt.Errorf("load client kp: %w", err) }
    pool := x509.NewCertPool()
    if !pool.AppendCertsFromPEM([]byte(creds.CACertPEM)) { return nil, fmt.Errorf("bad CA pem") }
    return &http.Client{
        Transport: &http.Transport{
            TLSClientConfig: &tls.Config{Certificates: []tls.Certificate{cert}, RootCAs: pool, MinVersion: tls.VersionTLS12},
        },
        Timeout: 15 * time.Second,
    }, nil
}
```

```go
package ca

import (
    "crypto/ecdsa"
    "crypto/elliptic"
    "crypto/rand"
    "crypto/x509"
    "crypto/x509/pkix"
    "encoding/pem"
    "fmt"
    "math/big"
    "time"

    "github.com/google/uuid"
)

type CA struct {
    CACertPEM []byte
    CAKeyPEM  []byte
    instanceID string
}

func Generate(instanceID string) (*CA, error) {
    priv, err := ecdsa.GenerateKey(elliptic.P256(), rand.Reader)
    if err != nil { return nil, err }
    serial, err := rand.Int(rand.Reader, new(big.Int).Lsh(big.NewInt(1), 128))
    if err != nil { return nil, err }

    tmpl := &x509.Certificate{
        SerialNumber: serial,
        Subject: pkix.Name{CommonName: "Triton Manage CA — " + instanceID},
        NotBefore: time.Now().Add(-time.Minute),
        NotAfter:  time.Now().Add(10 * 365 * 24 * time.Hour),
        KeyUsage:  x509.KeyUsageCertSign | x509.KeyUsageCRLSign | x509.KeyUsageDigitalSignature,
        BasicConstraintsValid: true, IsCA: true, MaxPathLenZero: true,
    }
    der, err := x509.CreateCertificate(rand.Reader, tmpl, tmpl, &priv.PublicKey, priv)
    if err != nil { return nil, err }
    keyDER, err := x509.MarshalPKCS8PrivateKey(priv)
    if err != nil { return nil, err }
    return &CA{
        CACertPEM: pem.EncodeToMemory(&pem.Block{Type: "CERTIFICATE", Bytes: der}),
        CAKeyPEM:  pem.EncodeToMemory(&pem.Block{Type: "PRIVATE KEY", Bytes: keyDER}),
        instanceID: instanceID,
    }, nil
}

// SignAgentCert returns (leafCertPEM, leafKeyPEM, error). Agent keypair is generated here
// because the agent ships the private key inside the bundle (operator hand-delivered).
func (c *CA) SignAgentCert(agentID uuid.UUID) ([]byte, []byte, error) {
    caCert, caKey, err := c.parse()
    if err != nil { return nil, nil, err }

    leafPriv, err := ecdsa.GenerateKey(elliptic.P256(), rand.Reader)
    if err != nil { return nil, nil, err }
    serial, _ := rand.Int(rand.Reader, new(big.Int).Lsh(big.NewInt(1), 128))

    tmpl := &x509.Certificate{
        SerialNumber: serial,
        Subject:      pkix.Name{CommonName: "agent:" + agentID.String()},
        NotBefore:    time.Now().Add(-time.Minute),
        NotAfter:     time.Now().Add(365 * 24 * time.Hour),
        KeyUsage:     x509.KeyUsageDigitalSignature | x509.KeyUsageKeyEncipherment,
        ExtKeyUsage:  []x509.ExtKeyUsage{x509.ExtKeyUsageClientAuth},
    }
    der, err := x509.CreateCertificate(rand.Reader, tmpl, caCert, &leafPriv.PublicKey, caKey)
    if err != nil { return nil, nil, err }
    leafKeyDER, err := x509.MarshalPKCS8PrivateKey(leafPriv)
    if err != nil { return nil, nil, err }
    return pem.EncodeToMemory(&pem.Block{Type: "CERTIFICATE", Bytes: der}),
        pem.EncodeToMemory(&pem.Block{Type: "PRIVATE KEY", Bytes: leafKeyDER}),
        nil
}

func (c *CA) parse() (*x509.Certificate, *ecdsa.PrivateKey, error) {
    cb, _ := pem.Decode(c.CACertPEM)
    if cb == nil { return nil, nil, fmt.Errorf("bad CA pem") }
    cc, err := x509.ParseCertificate(cb.Bytes)
    if err != nil { return nil, nil, err }
    kb, _ := pem.Decode(c.CAKeyPEM)
    if kb == nil { return nil, nil, fmt.Errorf("bad key pem") }
    rk, err := x509.ParsePKCS8PrivateKey(kb.Bytes)
    if err != nil { return nil, nil, err }
    kk, ok := rk.(*ecdsa.PrivateKey)
    if !ok { return nil, nil, fmt.Errorf("key not ecdsa") }
    return cc, kk, nil
}
```

```go
    return func(next http.Handler) http.Handler {
```
