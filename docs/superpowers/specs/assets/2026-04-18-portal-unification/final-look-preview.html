One token system · two themes

Both renders use the same HTML and component set — the difference is purely html[data-theme="dark|light"] on the root. You ship both, pick a default per-portal, and a topbar toggle lets each user override. The exported NACSA PDF always renders light.

1

Ship both · Slate dark default

Operators & engineers likely use this daily. Light stays one click away.

2

Ship both · Light default

Security officers & exec viewers are the primary audience. Dark stays one click away.

3

Per-portal default

Report → Light (officers), License & Manage → Slate (operators). Best of both.
